## VSCODE 확장 지점


### Code navigation

 

- Go to Definition (F12)

- Peek Definition (Alt+F12)

- Go to Implementation (Ctrl+F12)

- Go to References (Shift+F12)

- Go to Type Definition

  - 현재 요구사항과 관련한 식별자와 도움자의 속성자를 보여줌

- Go to Symbol in File (Ctrl+Shift+O)

- Go to Symbol in Workspace (Ctrl+T)

- Rename symbol (F2)

- InLay Hints

  - 식별자, 도움자의 정확한 fragment 를 표시

 

- Hover information (Ctrl+K, Ctrl+I)

  - 식별자, 도움자, 속성자 위에 popup box 로 정보 보여줌

- Signature help (Ctrl+Shift+Space)

  - (@), (=) 앞의 [] 에서 어떤 심볼이 있는지 보여줌

- Errors and warnings (Ctrl+Shift+M)

  - 링크가 누락되어 있거나, 잘못되어 있을 때

  - 요구사항, Testcase 템플릿을 보여줌
